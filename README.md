# CS168-zh

本项目基于[CS 168: The Modern Algorithmic Toolbox, Spring 2024](https://web.stanford.edu/class/cs168/)整理而成。

本课程将提供严谨且实践性强的课程 介绍构成的中心思想和算法 现代算法工具包的核心。重点将放在 理解高层次的理论直觉和原理 我们讨论的算法的基础，以及开发一个具体的了解何时以及如何实施和应用算法。 该课程将分为一系列为期一周的课程 调查；每周都会介绍一种算法思想，并且 讨论动机、理论基础和实践 该算法思想的应用。每个主题都会附有一个小型项目，学生将被指导通过实践。主题包括现代技术 在散列、降维、线性和凸规划中， 梯度下降和回归、采样和估计、压缩 传感、线性代数技术（主要成分 分析、奇异值分解、谱技术）以及差分隐私的介绍。

-  现代哈希
-  具有距离的数据（相似性搜索、最近邻、降维、LSH）
-  泛化和正则化
-  保持距离的压缩
-  线性代数技术：理解主成分分析
-  线性代数技术：了解奇异值分解和张量方法
-  谱图理论
-  采样和估算
-  傅立叶透视（和其他基础）
-  在线学习乘法权重和凸优化
-  隐私保护计算
